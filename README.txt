В данном коде планировалось реализовать функционал банкомата.
Было реализовано:
1. Снятие наличных средств (в двух валютах, с возможностью расширения).
    - ограничение по максимально возможной сумме снятия в день.
    - выдача купюр в наименьшем количестве (максимально крупными).
2. Проверка баланса (с двух валютных счетов, с возможностью расширения).
3. Проверка пользователя.
    - проверка наличия карты в банкомате.
    - проверка правильности пароля карты.

Даёт возможность пользователю, снять наличные средства в нескольких (имеющихся валютах).
Так же проверка своих счетов.
